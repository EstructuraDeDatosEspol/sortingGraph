Título
	Comparación de Algoritmos de Ordenamiento

Nombres de Autores:
	Miguel Angel Patiño San-Yeng
	Kenny Ariel Camba Torres 
	Steven Sebastian Araujo Moran 
	Kevin Axell Concha Regatto 

Descripción:
	Aplicación que compara los tiempos de ejecución de cuatro diferentes algoritmos de ordenamiento: 
	Merge Sort, Insertion Sort, Quick Sort, Stooge Sort. 

Archivo principal: 
	Main.java ubicado en src/espol/edu/ec/gui/

Dependencias:
	Java Development Kit versión 8
	NetBeans IDE 8.2
 	Librería externa: com.jfoenix:jfoenix:8.0.7 (src/espol/edu/ec/libs)

Instalación:
	Instalar NetBeans junto con el jdk8, fácilmente obtenibles desde la página principal 
	https://netbeans.org/ y https://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html 
	respectivamente, así también abrir el proyecto con NetBeans.

Uso:
	Para facilita su uso una vez instalado el jdk8 se podría acceder a la carpeta dist y ejecutar el archivo sortingGraph.jar
